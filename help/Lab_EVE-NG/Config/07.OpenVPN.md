1. Đặt/tinh chỉnh ip

```bash
sudo nano /etc/netplan/50-cloud-init.yaml
```

```bash
# nội dung file 50-cloud-init.yaml
network:
  version: 2
  renderer: networkd
  ethernets:
    ens33:
      dhcp4: false
      addresses:
        - 172.16.123.11/24
      routes:
        - to: default
          via: 172.16.123.1
      nameservers:
        addresses:
          - 172.16.123.1
          - 8.8.8.8

# ctrl + O  -> yes
# ctrl + x  -> để thoát
# apply IP vừa thay đổi bằng lệnh 'sudo netplan apply'
```
